## Coolify Bug - Empty Host() Label

**Problem:** Coolify generates `Host(``)` with empty domain in Traefik labels.
Bug affects all apps on this VPS (168.231.92.229).
Workaround created using YAML manual config.

### Workaround Applied (Traefik):

Created `/traefik/dynamic/zsg4s0gcog408c48wg0sg00s.yaml` inside coolify-proxy container:

```yaml
http:
  routers:
    pixelliber:
      rule: "Host(`pixelliberdigital.com`)"
      service: pixelliber
      tls:
        certResolver: letsencrypt
  services:
    pixelliber:
      loadBalancer:
        servers:
          - url: "http://10.0.0.236:3000"
```

Applied via:
```bash
docker exec coolify-proxy sh -c "cat > /traefik/dynamic/zsg4s0gcog408c48wg0sg00s.yaml << 'EOF'
yaml content
EOF"
docker exec coolify-proxy sh -c "docker kill -s HUP \$(docker ps -q --filter name=coolify-proxy)"
```
